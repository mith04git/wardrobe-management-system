Wardrobe Management System

Overview:-
This is a backend project built using Flask to manage wardrobe items and create outfit random outfit according to season.
It demonstrates REST APIs, database modeling, and backend logic using Python.

Tech Stack:-
- Python
- Flask
- SQLAlchemy
- MySQL
- REST APIs

Project Structure:-
wardrobe/
# app.py        (Main Flask application)
# models.py     (Database models)
# README.md     (Project documentation)
# .gitignore

 Features
- Backend APIs using Flask
- Database models using SQLAlchemy
- CRUD operations on wardrobe data
